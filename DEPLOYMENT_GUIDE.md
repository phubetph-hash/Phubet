# คู่มือการติดตั้งและใช้งานระบบที่ปรึกษานิสิต

## ภาพรวมระบบ

ระบบที่ปรึกษานิสิตเป็นระบบจัดการการขอคำปรึกษาจากอาจารย์ที่ปรึกษาสำหรับนิสิต โดยนิสิตสามารถส่งคำขอได้พร้อมกัน 5 คน และเมื่อมีอาจารย์ 1 คนรับเป็นที่ปรึกษา คำขออื่นๆ จะถูกยกเลิกอัตโนมัติ

## ข้อกำหนดระบบ

### ข้อกำหนดฮาร์ดแวร์
- **CPU**: Intel Core i3 หรือเทียบเท่า
- **RAM**: 4GB ขึ้นไป
- **Storage**: 10GB ขึ้นไป
- **Network**: การเชื่อมต่ออินเทอร์เน็ต

### ข้อกำหนดซอฟต์แวร์
- **Web Server**: Apache 2.4 หรือ Nginx
- **Database**: MySQL 5.7 หรือ MariaDB 10.3
- **PHP**: 7.4 หรือ 8.0
- **Node.js**: 16.x หรือ 18.x
- **OS**: Windows 10/11, macOS, หรือ Linux

## สรุป

ระบบที่ปรึกษานิสิตเป็นระบบที่ออกแบบมาเพื่อให้บริการการจัดการการขอคำปรึกษาจากอาจารย์ที่ปรึกษาสำหรับนิสิต ระบบนี้มีฟีเจอร์ที่ครบถ้วนและใช้งานง่าย สามารถรองรับการใช้งานได้อย่างมีประสิทธิภาพและปลอดภัย

การติดตั้งและใช้งานระบบนี้ไม่ซับซ้อน และมีคู่มือการใช้งานที่ชัดเจน หากมีปัญหาหรือข้อสงสัย สามารถติดต่อทีมสนับสนุนได้ตลอดเวลา
